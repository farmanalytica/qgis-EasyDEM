Principais recursos

Um conjunto direto de ferramentas para acesso, download e visualização de DEMs via Google Earth Engine no QGIS.

✓Autentique no GEE com seu ID de projeto do Google Cloud.
✓Selecione qualquer camada de polígonos do projeto QGIS como AOI.
✓Verificação automática de cobertura — apenas datasets disponíveis para a AOI aparecem na lista.
✓Mais de 20 datasets: Copernicus GLO-30, NASADEM, SRTM, ASTER GDEM, ALOS AW3D30, MERIT, USGS 3DEP e outros.
✓Resoluções de 0,5m a 1800m, cobrindo desde análises locais até estudos globais.
✓DEM adicionado ao projeto QGIS com rampa de cor Magma calibrada automaticamente.

Como funciona

Um fluxo de três etapas para sair do mapa ao DEM carregado no QGIS, sem ferramentas externas.

Autentique no GEE

Clique em "Authenticate on GEE", insira o ID do seu projeto Google Cloud e conclua a autenticação.

Selecione AOI e dataset

Escolha a camada de polígonos que define sua área de interesse. Os datasets disponíveis para a região carregam automaticamente.

Baixe e visualize

Selecione o dataset desejado e clique em "Download DEM". O raster é baixado e adicionado ao projeto com visualização pronta.

Catálogo de datasets

Datasets globais e regionais de fontes consolidadas, com resoluções de 0,5m a 1800m. A disponibilidade é verificada automaticamente para sua AOI.

✓Copernicus GLO-30 — DSM global a 30m derivado do TanDEM-X (ESA / Copernicus).
✓NASADEM & SRTM GL1 v003 — DEM global a 30m, cobertura de 60°N a 56°S (NASA / USGS).
✓ASTER GDEM v3 — DEM global a 30m de imagens ópticas estereoscópicas, 83°N a 83°S (NASA / METI).
✓ALOS AW3D30 v4.1 — DSM global a 30m do sensor PRISM a bordo do satélite ALOS (JAXA).
✓MERIT DEM & GMTED2010 & ETOPO1 — cobertura global a 90m, 250m e 1800m, incluindo batimetria oceânica.
✓USGS 3DEP 1m / 10m — LiDAR de alta resolução para os EUA (USGS).
✓Australia 5m DEM, ArcticDEM 2m, REMA 8m — datasets regionais de alta resolução para Austrália, Ártico e Antártida.
✓Netherlands AHN4 0,5m, France RGE ALTI 1m, England 1m — LiDAR nacional de alta densidade para Europa Ocidental.
✓GLOBathy — batimetria de máxima profundidade para 1,4 milhão de lagos e reservatórios globais (30m).
